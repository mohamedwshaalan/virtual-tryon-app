# virtual-tryon-app
Github Repository for Thesis Project

# Requirements
- Python 3.6+
- Flask
- Flutter

# Backend Setup
```
> git clone https://github.com/mohamedwshaalan/virtual-tryon-app.git
> cd virtual-tryon-app
> py -3 -m venv .venv  
> .venv\Scripts\activate
> pip install -r requirements.txt
> flask --app main run
```
